# Codex_UI设计整改提示词

你的任务是把《大梁武侠TRPG》APP做得稳定、有序、好看，而不是自由发挥。

## 必读文件
1. `DESIGN.md`
2. `UI_PAGE_BLUEPRINTS.md`
3. `UI_COMPONENT_REGISTRY.md`
4. `design_tokens.json`
5. `大梁武侠TRPG_v0.42整改总控计划包.xlsx` 中的 `UI设计管控方案`、`APP页面蓝图`、`GitHub_UI工具评估`

## UI原则
- 采用 React + TypeScript + Tailwind + shadcn/ui + lucide-react。
- 左侧导航 + 顶部状态栏 + 主内容卡片区 + 右侧详情/日志抽屉。
- 风格：宣纸米白、墨色、黛青、朱砂、金石灰。
- 信息密度中等，表格清楚，按钮少而明确。
- 每次只做一个页面或一个组件组。
- 每页实现后必须截图验收，低于80分重做。

## 禁止
- 不要使用随机渐变和杂乱色块。
- 不要每页一套风格。
- 不要没有蓝图就写代码。
- 不要一次生成整个APP。
