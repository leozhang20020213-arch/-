## 3. 测试方式摘要

### 普通玩家体验测试

创建了以下普通玩家智能体：

- 新手玩家
- 剧情玩家
- 战斗玩家
- 普通构筑玩家
- 随性玩家
- 机动型普通玩家

并创建：

- 普通 DM
- 监管记录器 / supervisor

测试重点是普通玩家是否玩得下去、玩得开心，而不是让所有智能体充当规则审计员。

### 严格长战役测试

三段不同长战役，每段 6 个循环：

1. `SLC01 京畿暗账长团`
   - 主题：调查、官府、人情、把柄。
2. `SLC02 北地断碑长团`

3. `SLC03 南海潮生长团`
   - 主题：商路、名望、家庭、产业、海盗。

每个循环包含：

- 战役目标
- 情景场景
- 冲突/战斗
- 奖励与代价
- 三阶段休整
- 岁月/江湖动态事件

- 玩家反馈
- DM 负担记录
- 规则使用率记录

## 4. 总体结论

规则目前可以开测试团，但不建议直接进入正式大规模长团。

核心框架能跑：

- 情景检定能推动调查、交涉、潜入、追逐。
- 三槽分配确实能制造取舍。
- 危机值能制造压力。
- 失败不堵关键线索是有效原则。
- 武器差异在得利门槛清楚时能被玩家感知。
- 成长检定让玩家期待下一局。

但完整战役和长团会放大以下问题：

- 成功但危机持续上涨，会造成行动焦虑。
- DM 长团记录负担很高。
- 战役得利值有用，但容易漏记。
- 普通敌人频繁反击会拖慢并削弱爽感。
- 武学谱、内功成长、连招没有卡片入口时不会自然出现。
- 名望、人情、情报、把柄、家庭、产业如果没有消费入口，会退化成背景记录。

## 5. 最稳定复现的不好玩问题

### 5.1 成功但像受罚

标签：

- 【太惩罚】
- 【数值直觉不对】

现象：

- 玩家多次成功，但因为代价槽低，危机持续上涨。
- 单场还能接受，长团中会累积成“行动焦虑”。
- 玩家开始觉得“少行动可能更安全”。

玩家原话摘录：

- “我们又成功了，为什么越来越糟？”
- “我觉得我们很努力，但每次代价都把好事打折。”
- “成功时先夸我做到了，再说麻烦，我会舒服点。”

建议：

- 先补主持话术，不立即改阈值。
- 固定口径：先说玩家成功获得什么，再说代价如何改变局势。
- 做数学模拟后再判断是否调整代价阈值。

### 5.2 普通敌人频繁反击不爽

标签：

- 【等待过长】
- 【不够爽】
- 【太惩罚】

现象：

- 小兵频繁因破绽2反击，会让战斗变黏、变慢。
- 玩家接受高手和 Boss 正式反击，甚至觉得有武侠味。
- 玩家不希望普通小兵也像高手一样频繁反打。

建议：

- 必须修敌人模板口径：
  - 普通敌人：默认削得利或轻反应。
  - 精英敌人：简化反击。
  - Boss/高手：正式反击。
- 反击动作默认不再触发新的反击链，除非 Boss 条目明确例外。

### 5.3 武器特性低得利时挫败

标签：

- 【不够爽】
- 【看不懂】

现象：

- 长枪、暗器等依赖得利2触发特性的武器，得利1时玩家觉得“动作说了但没发生”。
- 新手需要 DM 告诉他“想击退就要得利2”。

建议：

- 每件武器卡写明：
  - 得利1能做什么。
  - 得利2触发什么特性。
  - 得利3有什么额外效果。
  - 推荐分槽策略。

### 5.4 长团资源没有消费入口

标签：

- 【选择无意义】

现象：

- 名望、人情、情报、把柄、家庭、产业能被记录。
- 但如果没有明确消费入口，玩家不会主动使用。
- 它们会变成背景设定，而不是战役资源。

建议：

- 每种长期资源至少给 3 个默认消费入口。
- 例如：
  - 名望：请求地方协助、获得高层委托、压低普通敌人士气。
  - 人情：换通行、换线索、换安全撤离。
  - 情报：降低 DC、提前知道敌人、开启备选路线。
  - 把柄：迫使 NPC 让步、换证词、阻止势力干涉。

## 6. 很少使用的规则

来自 `12_严格长战役规则使用率分析.xlsx` 的低使用项：

| 规则 | 机会 | 触发 | 使用率 | 观察 |
| --- | ---: | ---: | ---: | --- |
| 战役得利达到阈值触发解锁 | 18 | 0 | 0% | DM 容易漏记，玩家感知弱 |
| 协助行动 | 18 | 0 | 0% | 玩家更倾向自己行动，缺少协助激励 |
| 强行尝试 | 18 | 0 | 0% | 规则存在，但玩家不常主动使用 |
| 稳控机会 | 18 | 0 | 0% | 触发时机抽象，DM 不确定何时给 |
| 非杀伤攻击/制服 | 18 | 0 | 0% | 有机会但玩家常直接击败 |
| 环境互动动作 | 18 | 0 | 0% | 随性玩家会问，但规则入口不清 |
| 轻功/Pos/追逐位势 | 18 | 0 | 0% | DM 倾向抽象处理，不愿精算 |
| 武学谱修行 | 多次 | 0 | 低 | 没有招式卡时不会自然触发 |
| 内功成长 | 多次 | 0 | 低 | 没有内功卡和成长目标时不会自然触发 |

重点判断：

- 这些规则不一定应删除。
- 其中不少是“规则有用，但缺少入口、卡片或示例”。
- 最危险的是战役得利值：它本来能缓解危机压力，但目前太容易漏记。

## 7. DM 负担在哪里

### 7.1 长团记录负担

DM 需要同时记录：

- 危机
- 解密
- 战役得利
- 名望
- 人情
- 情报
- 把柄
- 伤势
- 仇敌
- 弟子
- 家庭
- 产业
- 岁月事件
- 下一钩子

结论：

- 这已经超出普通纸笔随手记的舒适范围。
- 必须提供长团记录表，或建议 APP/表格辅助。

### 7.2 裁定负担

DM 经常需要判断：

- 什么算关键线索行动。
- 什么时候推进解密值。
- 什么时候给稳控机会。
- 成功但代价高时给哪类代价。
- 得利1/2/3能兑现多强效果。
- 非杀伤攻击算攻击、控制还是环境互动。
- 普通敌人破绽2到底反击、削得利还是断势。

结论：

- 这些不是“DM 不会跑”，而是规则缺少默认口径和示例。

## 8. 必须修

### 8.1 普通敌人破绽兑现口径

当前问题：

- 普通敌人频繁反击会明显拖慢战斗。
- 玩家不喜欢“小兵像高手一样反打”。

建议写法：

```text
普通敌人：破绽2默认削得利或轻反应，不进行正式反击。
精英敌人：可使用简化反击。
Boss/高手：可使用正式反击。
反击动作默认不再触发新的反击链，除非条目明确写明。
```

### 8.2 熟练档位与武学容量文本冲突

此前测试已发现：

- 熟练档位名称/骰阶口径前后不一致。
- 武学容量存在双口径。

建议：

- v0.42 必须统一。
- 否则构筑玩家和 DM 都会在开卡和成长阶段卡住。

## 9. 优先补示例

建议优先补：

1. 高代价成功话术。
2. 得利1/2/3跨场景强度表。
3. DM 长团三轨记录表。
4. 战役得利解锁菜单。
5. 场景行动菜单。
6. 协助行动默认口径。
7. 稳控机会发放与使用例子。
8. 非杀伤攻击和环境互动菜单。
9. 武器卡。
10. 预设角色武学招式卡。
11. 内功卡。
12. 名望/人情/情报/把柄消费入口。

## 10. 暂缓项

暂不建议立刻改：

1. 代价等级阈值。
   - 玩家体感稳定觉得太惩罚，但需要数学模拟确认。
2. 破绽等级阈值。
   - 先改敌人兑现口径再复测。
3. 暗器/长枪是否数值过弱。
   - 先补武器卡和得利1效果。
4. 稳控机会是否改成通用动作。
   - 目前更像缺示例，不一定要改机制。

## 12. v0.42 优先修改方案

### 第一优先级：能否顺畅开团

1. 修熟练档位与武学容量冲突。
2. 修普通敌人破绽兑现口径。
3. 补 DM 三轨记录表。
4. 补高代价成功话术。
5. 补得利1/2/3强度表。

### 第二优先级：让长团资源有意义

1. 补名望、人情、情报、把柄消费入口。
2. 补家庭、弟子、产业如何改变场景权限。
3. 补战役得利解锁菜单。
4. 补休整行动收益对比。

### 第三优先级：让武学系统进入游戏

1. 每个预设角色给招式卡。
2. 每个预设角色给内功卡。
3. 写默认连招路线。
4. 明确普通武器动作、散招、单招、武学谱招式的区别。

## 13. 希望 ChatGPT 重点审阅的问题

请重点审阅以下判断是否成立：

1. 是否同意“普通敌人频繁反击”应作为必须修。
2. 是否同意“成功但危机上涨”先补话术和模拟，而不是马上改阈值。
3. 是否同意“战役得利值”不应删，而应补记录表和解锁菜单。
4. 是否同意“武学/内功低使用”主要是缺少卡片入口，而不是机制本身无用。
5. 是否同意长团资源需要明确消费入口，否则会变成背景记录。
6. 是否还有更高优先级的必须修项目。

## 14. 结论

当前规则适合继续进行 v0.42 修订前的测试，但不建议直接作为正式公开长团版本。

核心可用：

- 情景推进。
- 三槽分配。
- 危机压力。
- 失败不堵线索。
- 武器差异。
- 成长检定。

核心风险：

- DM 记录压力。
- 高代价成功的负面体感。
- 小兵反击拖战斗。
- 长团资源没有消费入口。
- 武学与内功缺少上桌卡片。

建议进入下一轮：

1. 做数值模拟。
2. 出 v0.42 修改草案。
3. 针对本报告列出的“必须修”和“补示例”项进行规则书结构修订。
