# 大梁江湖TRPG — 游玩交互设计 v1.0

> 设计目标：像杀戮尖塔一样流畅的卡牌交互 + 跑团模拟器的叙事沉浸  
> 核心理念：每个操作 ≤2步（选牌→选目标/放骰子），所有复杂信息在抽屉/浮窗

---

## 一、玩家心流模型

```
开局 → 了解场面 → 看手牌 → 选招式 → 配气骰 → 选目标 → 看结算 → 对手回应 → 看落果
  ↑                                                                              ↓
  └──────────────────────── 下一轮 ← 调息/返照 ←────────────────────────────┘
```

**每一步都有清晰的视觉焦点，不分散注意力**。

---

## 二、交锋回合完整操作流

### Step 1: 回合开始 — 看场面

```
[画面状态]
- 中央战场显示所有角色位置+距离线
- 左侧你的角色卡高亮，脉动边框
- 底栏手牌区所有可用招式卡亮起

[系统提示]
"第3轮 · 你的回合"  — 浮动文字，2秒后淡出

[玩家看到的]
- 自己：沈青 气血■■■■ 合势
- 对手：短兵客 气血■■■ 阳盛（行为提示："准备截击"）
- 手牌：破浪横刀(亮)、燕回侧步(亮)、截腕挑锋(暗-"需要阴盛")
```

### Step 2: 选招式 — 点击手牌

```
[玩家操作]
点击手牌中的 "破浪横刀"

[立即反馈]
1. 该卡牌弹起，金边高亮
2. 气海中符合条件的骰子自动高亮（黄色边框脉动）
3. 不符合条件的骰子变灰
4. 招式详情浮窗出现：
   ┌─────────────────────┐
   │ 破浪横刀  起式·少阳  │
   │ 最低投入：2骰        │
   │ 气性要求：至少1阳     │
   │ 势条件：宽势 ✓(合势) │
   │ 装备许可：需刀 ✓(环首刀)│
   │ 基础效果：造成气血4点 │
   │ 触发：阳值≥7→追加3点 │
   │ 风险：阴阳差≥5→破口1 │
   └─────────────────────┘

5. 目标选择提示："请选择目标"
6. 战场上可选目标高亮（金色光圈）
7. 不可选目标变灰（带原因）

[如果选了不可用的招式]
- 卡片点击无效
- 灰色遮罩上浮动原因文字："需要阳盛（当前合势）"
- 持续1.5秒后淡出
```

### Step 3: 配气骰 — 拖拽到阴阳槽

```
[招式已选中，目标待选]
此时玩家可以做两件事（顺序无所谓）：
A. 选目标
B. 配气骰

[B. 配气骰操作]
气骰操作区放大高亮，提示"请配置气骰"

1. 从气海拖一枚骰子（按住左键）
   - 骰子跟随鼠标，带半透明拖影
   - 有效投放区（阴槽/阳槽）亮绿色边框
   - 无效投放区（如该骰是阴骰但阳槽只缺阳骰）显示红色边框

2. 放入阴槽
   - 骰子落入槽中，带弹性动画
   - 槽位计数器更新："阴值: 4"
   - 气海中该骰子消失

3. 放入阳槽
   - 同上，"阳值: 6"

4. 如果配置满足成招门槛
   - 锁气区显示合计："2骰 阴4阳6 · 合值10"
   - 招式卡上的触发条件实时更新：
     "阳值≥7 ✓ (6) → 未触发"
     "合值≥10 ✓ → 触发：目标退距1档"
     "阴阳差≥5 ✗ (2) → 不触发"

5. 可以拖回来重新配（在宣言确认之前）
   - 从槽中拖回气海即可

[拖拽防误操作]
- 如果试图放入不匹配的槽（如把阴骰放入仅缺阳的槽）→ 骰子弹回气海 + 抖动
- 已经锁气确认后不可拖回
```

### Step 4: 选目标 + 宣言

```
[A. 选目标]
点击战场上的目标 或 右侧敌人列中的目标卡片

- 点击目标 → 目标高亮（红色光圈）
- 从玩家到目标画一条虚线箭头
- 目标卡片展开显示：当前气血/护体/状态/行为提示

[确认宣言]
阴阳槽配好 + 目标选好 → 底栏"宣言"按钮亮起

点击 [宣言]
1. 宣言动画：
   - 招式卡从手牌飞出 → 战场中央
   - 锁气区骰子锁定（不可拖回）
   - 画面微微暗角，焦点聚到战场
2. 进入截击窗口

[如果被截击]
- 画面短暂震动
- 宣言卡裂开效果
- 系统提示："短兵客 截击！截腕挑锋 → 你的锁气中1枚阳骰移入息库"
- 如果宣言被取消 → 招式卡破碎动画 → 气骰入息库 → 回到手牌选择
```

### Step 5: 截击窗口（对手行动时）

```
[系统提示]"短兵客正在宣言「回潮压刃」→ 沈青   你有2秒可以截击"

1. 画面顶部出现倒计时环（默认15秒，可配置）
2. 手牌区浮现可用的截击招式（带蓝色边框）
3. 不可用的截击灰色显示

[如果你有可用的截击]
点击截击卡 → 选气骰 → 确认截击

[如果你选择不截击]
倒计时结束自动跳过 → 进入成招结算

[DM可配置]
- 截击窗口时长
- 是否显示倒计时
- 是否允许跳过
```

### Step 6: 成招结算 — 看演出

```
[宣言未被截击/截击不影响成招]
进入成招阶段（自动，约2秒演出）

1. 招式卡放大 → 战场中央
2. 锁气骰子依次翻面显示点数
3. 阴槽阳槽数值滚动上升
4. 触发效果逐条弹出：
   "⚔️ 造成气血4点"  
   "⚡ 阳值=6 → 未达到7 → 不触发追加"
   "📐 合值=10 → 触发：目标退距1档"
   "⚠️ 阴阳差=2 → 未达到5 → 安全"

5. 伤害数字从目标头上浮起，红色，800ms淡出
```

### Step 7: 应招窗口（你可以回应对手的伤害）

```
[如果你/同伴成为目标，且伤害即将落果]
你可以用应招减轻伤害

1. 手牌区浮现可用应招（绿色边框）
2. 点击应招 → 配气骰 → 确认
3. 应招效果结算（减伤/护人/转移）

[如果你选择不应招]
倒计时结束 → 伤害落果
```

### Step 8: 落果 + 回合结束

```
[落果]
1. 最终伤害写进角色卡
2. 状态图标弹出+贴到角色卡上
3. 势变化：角色卡边框颜色切换+势名浮动
4. 已用骰子飘入息库
5. 回合结束

[回合结束检查]
- 状态衰减动画（如"流血-1"在角色上方浮动后消失）
- 下一行动者高亮
```

## 四、手牌系统设计（类杀戮尖塔）

### 4.1 手牌区布局

```
┌──────────────────────────────────────────────────────────────┐
│ ← 手牌可左右滚动 →                     [调息] [返照] [结束]  │
│                                                              │
│ ┌──────┐┌──────┐┌──────┐┌──────┐┌──────┐┌──────┐         │
│ │破浪  ││燕回  ││截腕  ││回身  ││听痕  ││收刀  │         │
│ │横刀  ││侧步  ││挑锋  ││护肩  ││辨路  ││归息  │         │
│ │2骰·阳││2骰·阴││2骰·阴││2骰·阴││2骰·阴││2骰·任│         │
│ │宽势✓ ││宽势✓ ││宽势  ││宽势✓ ││法门  ││宽势  │         │
│ │刀✓   ││无装备││空手✓ ││可介入││无装备││需刀  │         │
│ │⚔4  ││🦶移步││🔒锁气││🛡抵消││🔍解密││♻回气│         │
│ └──────┘└──────┘└──────┘└──────┘└──────┘└──────┘         │
│  可用   可用   不可用  可用    可用    不可用               │
│                     (需阴盛)                   (需合势/圆融)│
└──────────────────────────────────────────────────────────────┘
```

### 4.2 招式卡设计

每张招式卡 = 一张"卡牌"，含：

```
┌─────────────────┐
│ ⚔️ 破浪横刀       │  ← 招式名 + 图标
│ 起式 · 少阳      │  ← 式位 + 路数标签
│                 │
│ 基础：气血-4     │  ← 基础效果（必定触发）
│                 │
│ 阳值≥7 → +3     │  ← 主槽触发
│ 合值≥10 → 退距   │  ← 合值触发
│ 差≥5 → 破口1    │  ← 风险
│                 │
│ 2骰 · 至少1阳   │  ← 最低投入+气性
│ 宽势 · 需刀     │  ← 门槛条件
│                 │
│ [挂载：截击✓ 应招✓]│ ← 响应挂载标记
└─────────────────┘
  160×220px

卡牌状态：
- 可用：正常色，可点击，悬停抬起
- 选中：金边+放大到180×240
- 不可用：灰色50%透明+不可用原因文字
- 冷却中：暗色+回转图标
```

### 4.3 手牌数量控制

- 手牌区同时可见约6-7张
- 如果角色有更多招式，左右箭头滚动
- 便行招式（移步/整身/取物/闭息/调息）固定在右侧独立区域
- 截击/应招招式在需要时才浮现（截击窗口/应招窗口打开时）

## 五、操作优先级和智能辅助

### 5.1 智能高亮

系统自动帮玩家判断：
- **点选招式后**：高亮合规气骰 + 合规目标（一个动作最多两个判断）
- **气骰不够时**：招式卡变灰，提示"需要2骰，气海仅1骰"
- **势不对时**：招式卡变灰，提示"需要阳盛（当前合势）"
- **缺装备时**：招式卡变灰，提示"需要刀（已脱手）"

### 5.2 快捷操作

| 操作 | 快捷方式 |
|---|---|
| 快速宣言（不用选骰） | 右键招式卡 → "自动配骰" → 系统自动选最优骰子放入阴阳槽 |
| 重复上轮招式 | 双击上轮用过的招式卡 |
| 快速调息 | 点击息库 → "调息"→ 系统自动取回所有常规骰 |
| 跳过响应 | 空格键 / 点击"跳过"|
| 打开抽屉 | F1角色 F2背包 F3武艺 F4日志 F5规则 |
| 确认/宣言 | Enter |
| 取消选择 | Esc |

### 5.3 防误操作

- 宣言前必须同时：招式已选 + 气骰已配 + 目标已选
- 宣言按钮在这三个条件满足前一直是灰色不可点击
- 已经宣言进入截击窗口后不可撤回（只能被截击取消）
- 关闭应用的确认弹窗

## 六、DM操作界面

DM看到的和玩家一样，但有额外控制：

### DM独有的按钮/面板

1. **场景控制面板**（右侧固定）
   - [新场景] — 进入新场景，所有气骰从气池重投
   - [情景→交锋] — 将当前情景切换为交锋，弹出确认
   - [交锋→情景] — 结束交锋回到情景
   - [手动裁定] — 弹出输入框，写入日志

2. **敌人控制**
   - 点击敌人 → 手牌切换为该敌人的招式
   - 敌人招式卡底部有AI行为提示标注
   - [自动] — 让AI自动选择敌人的行动（按behaviorHint）

3. **隐藏信息**
   - DM抽屉中可看到敌人的 hiddenGoal/behaviorHint/hiddenStatuses
   - 玩家看不到的信息在DM面板用红色虚线边框标记

4. **广播**
   - [广播消息] — 向所有玩家发送系统提示
   - 危机值调整、解密值推进、场景许可授予

### DM的回合流

```
1. 查看场面（所有角色位置/状态）
2. 如果有NPC该行动：
   a. 点击NPC
   b. 选招式（或按 [自动] 让AI选）
   c. 确认宣言
3. 玩家行动时：等待宣言 → 可选截击/应招
4. 落果后检查 → 下一轮或切回情景
```

## 七、情景模式交互

情景模式像"视觉小说+点击探索"：

### 操作
- **点击场景插图上的热点** → 触发调查/互动
- **点击右侧NPC** → 展开对话选项
- **点击物件** → 检查/拾取/使用
- **底栏行动按钮** → 调查/对话/潜行/使用物品/进入交锋

### 情景→交锋的过渡

```
[DM或系统判定]"此行动将进入交锋"
    ↓
[画面效果] 暗角加深 → "交锋"毛笔字浮现 → 
场景区收缩 → 角色列/手牌/气骰区按顺序滑入
    ↓
[进入交锋·准备阶段]
```

## 八、特殊交互场景

### 8.1 开局 → 选角色

大厅选角像选角色（像杀戮尖塔选角色页面）：
- 3个快速开卡包并排显示
- 选中后展开六根雷达图 + 关键数值 + 初始武艺预览
- 下方[开始游戏]按钮

### 8.2 濒死

角色气血归零时：
- 角色卡变红闪烁
- 系统提示"沈青进入濒死状态！"
- 手牌区全部变灰（濒死不能用正式出手）
- 仅剩：返照按钮亮起 / 求救选项 / 少数随手便行
- 队友可用的救人招式/药物自动高亮

### 8.3 调息/返照

**调息**：
- 气海未空 → 点击息库中骰子 → [调息]按钮 → 3D骰子投掷遮罩 → 重掷结果回气海
- 消耗一次出手

**返照**：
- 气海已空（断气）→ 返照按钮自动亮起
- 点击 → 自动取回息库中最低骰 → 不重掷 → 入气海
- 不消耗出手（随手便行）

### 8.4 使用物品

- 打开背包抽屉 → 点击药物/器具 → [使用]按钮
- 立即生效：加气血/清状态/临时气骰出现在临气区
- 背包中物品数量-1
- 如果物品是装备 → [装备]按钮 → 属性加成立即加到表属性

## 十、新手引导流程

首次打开游戏时：

```
1. [欢迎] 大梁江湖TRPG欢迎你
2. [选角] "这是你的角色卡。六根是根基，内功盘决定你的打法。"
   → 引导点击"快速开卡包-镖局刀客"
3. [进场景] "你来到了桥陵镇。这是场景画面，点击热点可以调查。"
   → 引导点击场景中的茶棚
4. [进交锋] "有人拔刀了！准备交锋。"
5. [第一回合] 
   → "这是你的手牌。点击'破浪横刀'。" 
   → "把骰子拖到槽里。至少1阳。" 
   → "选目标。" 
   → "点击宣言。"
6. [被截击] "对手截击了！你的宣言被取消。"
7. [第二回合]
   → "试试调息，回收你的骰子。"
   → "再用'燕回侧步'配合应招。"
8. [结束] "恭喜完成第一场交锋！你现在可以自由探索了。"
```

每个引导步骤都是一个半透明遮罩+高亮目标区域+文字说明，点击即进入下一步。
