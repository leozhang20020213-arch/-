# 大梁江湖TRPG — 新手融入设计

> 设计原则：像游戏选角一样开卡，像新手村一样学规则，像解锁技能一样成长。
> 不让玩家读文档，让玩家玩会。

---

## 一、总体思路

### 不是这样的 ❌
```
[表单] 姓名:___ 六根:___ ___ ___ ___ ___ ___ 内功:___
→ 新玩家看一眼就走了
```

### 是这样的 ✅
```
[画面] 雨夜客栈，三个人物剪影
"雨夜，桥陵镇。你为何而来？"
→ 选一个 → 自动生成角色 → 即刻进入场景 → 边玩边学
```

### 阶段2：初入江湖 = 第一个场景（2分钟）

进入场景后不是在空房间里站着，而是立刻有NPC招呼：

```
[场景画面：桥头茶棚，雨夜]
[叙述] "茶棚掌柜看你淋得透湿，递来一碗热茶。"

[系统提示] ← 这里不是弹窗，而是一封"书信"的样式
┌──────────────────────────┐
│ 你可以在场景中：          │
│ · 点击场景中的物品调查    │
│ · 点击右侧的人物对话      │
│ · 底栏是你可用的法门      │
│                          │
│ 试试点击"茶棚掌柜"对话    │  ← 引导高亮
└──────────────────────────┘
```

### 阶段3：初战 = 学会交锋（3分钟）

场景中自然引出冲突：
- 不是"现在进入教程战斗" 
- 而是"门外传来脚步声，几个黑影逼近"
- DM/系统："他们好像来者不善。你打算怎么做？"

```
画面自然切入交锋模式：
1. 手牌区自动展开
2. 系统提示（书信样式）："这是你的招式手牌。点击一张试试。"
3. 玩家点击→引导配骰→引导选目标→宣言
4. 第一战对手很弱（1个喽啰），保证玩家赢
5. 胜利后：角色卡弹出"成长"提示
```

## 三、实现层：添加三个新路由

### 路由设计

```
/                → 启动画面
/select          → 角色选择（三选一）
/create          → 场景创建/加入
/scene           → 情景模式
/combat          → 交锋模式
```

### `/select` — 角色选择页

核心组件 `<CharacterSelect>`：
- 展示3个预设角色卡
- 悬停展开六根+关键招式
- 点击选中→高亮+展开详情抽屉
- 确认→进入角色微调（姓名可改，其余自动生成）→进场景

### 融入技巧

1. **不要"教程"按钮**，而是第一次进场景时自然触发引导提示
2. **引导提示不是弹窗**，是画面中浮出的书信/锦囊样式，可关闭可跳过
3. **招式卡悬停自动显示说明**，新玩家不需要记住任何规则
4. **不可用的操作变灰+原因**，让玩家自然理解限制条件
5. **第一场战斗难度极低**，让玩家在安全环境里尝试各种操作
6. **失败不惩罚**：新手第一场战斗可以重来

## 四、开卡流程（对应规则书第17章）

### 快速开卡（推荐新玩家）

```
选身份 → 分配六根 → 选初始内功 → 自动算表属性 → 自动生成气骰
→ 选初始武艺（3选2或自动配）→ 自动配装备 → 确认角色名 → 完成
```

每一步都是一张"卡牌翻转"的动画：
1. 身份卡翻牌 → 显示六根
2. 六根卡翻牌 → 显示内功盘
3. 内功盘卡翻牌 → 显示表属性和气骰
4. 完成卡 → 显示完整角色卡

### 自定义开卡（高级玩家）

```
1. 24点分配六根（每根1-9，单根开局最高6）
   默认推荐：6/5/4/4/3/2
   UI是一个六边形雷达图，可以拖动顶点调整数值

2. 选初始内功（3-5门可选）
   每门内功卡片显示：占窍/读取六根/表属性倾向/被动

3. 选初始武艺（从3-5条中选2-3条）
   至少：1条常用外功+1条自保/应招+1条非战斗法门

4. 自动配装备（根据武艺的装备许可）

5. 确认角色名+出身+性格
```

## 五、引导系统实现

不用全局状态机，用 localStorage 存 `newPlayerFlags`：

```typescript
interface NewPlayerFlags {
  hasSeenCharacterSelect: boolean;   // 已选角
  hasCompletedFirstScene: boolean;   // 已完成第一个场景交互
  hasCompletedFirstCombat: boolean;  // 已完成第一场战斗
  hasUsedIntercept: boolean;         // 已使用过截击
  hasUsedReact: boolean;             // 已使用过应招
  hasUsedBreathRegulation: boolean;  // 已使用过调息
  hasUsedReflection: boolean;        // 已使用过返照
  skipAllTips: boolean;              // 跳过所有引导
}
```

每完成一个里程碑 → 标记已达成 → 不再提示同类引导。

引导提示样式：画面右下角浮出锦囊，带金色丝线装饰，2秒后自动缩小到角落的问号图标，点击可重新展开。
