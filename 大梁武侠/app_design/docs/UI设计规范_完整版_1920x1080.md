# 大梁江湖TRPG — UI设计规范 v1.0

> 画布：1920×1080px 单屏  
> 风格：武侠卷轴/案桌质感 — 深木色底 + 宣纸面板 + 朱砂点缀 + 墨色文字  
> 交互：所有二级内容走抽屉滑出，主界面永远不跳页  

---

### 1.2 字体系统

| 层级 | 字体 | 大小 | 用途 |
|---|---|---|---|
| 主标题 | 思源宋体/Noto Serif CJK Bold | 28px | 场景名/角色名 |
| 面板标题 | 思源宋体 Bold | 20px | 卡片标题 |
| 正文 | 思源宋体 Regular | 16px | 描述文字 |
| 小字 | 思源宋体 Regular | 13px | 说明/标签 |
| 数字 | 思源宋体 Bold | 18px | 数值显示 |
| 骰子数字 | Inter/等宽 | 14px | 气骰点数 |
| 系统提示 | 思源宋体 Regular | 14px | 日志/系统消息 |

### 1.3 通用交互规则

- **抽屉**：所有详情内容（角色卡/背包/武艺/日志/规则）统一从右侧滑入，宽度520px，带10px左边阴影
- **提示框**：屏幕中央弹出，400×200px，半透明深底+宣纸卡片
- **确认窗**：同提示框，增加确认/取消双按钮
- **悬停**：按钮+高亮边框；卡片+微微抬起（translateY(-2px)）+ 阴影加深
- **过渡**：所有面板变化200ms ease-out；抽屉滑入300ms ease-out；骰子投掷实时物理
- **音效占位**：每个交互标注音效类型，实际音频后期替换

### 区域详解

**左侧：场景插图区** (0,60 ~ 960,900)
- 场景美术图占位，可叠加可交互热点
- 热点：鼠标悬停高亮，点击触发行动
- 动景标记区（火势/雨势/追兵距离）

**右侧上：场景目标 & 人物** (960,60 ~ 1920,450)
- 场景目标卡片：200×80px
- 可见人物列表：每个80px高，含姓名/距离/势/简要状态
- 可互动物件列表：每个48px高

**右侧下：动景 & 轨道** (960,450 ~ 1920,900)
- 危机值轨道：横向条+名称+数值
- 解密值轨道：横向条+名称+数值
- 当前许可列表

**底部：叙述区 + 行动栏** (0,900 ~ 1920,1080)
- 叙述区：300px高，滚动文本
- 行动按钮栏：56px高，水平排列

### 交互
- 点击场景热点 → 触发情景行动
- 点击NPC → 展开对话/交互选项
- 点击物件 → 显示详情+可用行动
- 点击"进入交锋" → 确认弹窗 → 切到视图3
