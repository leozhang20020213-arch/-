## 二、时点系统（隐藏式）

不显示6个阶段圆点。像卡牌游戏一样：
- 玩家回合 → 手牌亮起 → 可选招式
- 选招式后 → 自动进入"宣言"时点 → 目标线高亮
- 对手截击 → 截击卡从手牌区弹出（仅在该时点可见）
- 成招后 → 应招卡弹出（仅在该时点可见）
- 落果后 → 手牌恢复

**视觉暗示**（替代阶段圆点）：
- 手牌区上方一条2px细线，颜色随阶段变化：
  - 宣言中 = 金色
  - 截击窗口 = 蓝色闪烁
  - 应招窗口 = 绿色闪烁
  - 落果 = 红色
  - 空闲 = 无/暗色

## 四、抽屉行为

1. **展开位置**：居中展开（在目标线区域下方）
2. **多抽屉**：多个抽屉水平排列，每个最小300px宽
3. **拖拽调整**：抽屉右边缘可拖拽调整宽度（200-600px）
4. **关闭**：点击抽屉外的区域 / 再次点击触发按钮
5. **层级**：最后打开的抽屉在最前面

## 五、玩家视角 vs DM视角

| 元素 | 玩家视角 | DM视角 |
|---|---|---|
| 队友角色卡 | 显示（左侧） | 显示所有 |
| 敌人角色卡 | 显示公开信息 | 显示全信息+隐藏目标 |
| 敌人隐藏状态 | 不显示 | 红色虚线边框标记 |
| DM按钮 | 不显示 | 左侧竖排显示 |
| 手动裁定 | 不可用 | 可用 |
| 场景管理 | 不可用 | 可用 |
