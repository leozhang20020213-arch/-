# 资源区补全 — 息库与临气区

**日期**: 2026-06-25
**分支**: `dice-system-first-pass`
**状态**: ✅ 完成

## 改动摘要

本轮只改底部气骰资源区，补出"息库"和"临气区"两个此前只在规则中存在、UI 中缺席的区块。不改左侧/右侧/全局美术。

---

## 一、临气区 (TemporaryQiPool) — 从"有骰才显"改为"始终可见"

**改动前**：`TemporaryQiPool` 在没有 temp dice 时直接返回 `null`，UI 中完全消失。

**改动后**：
- 始终渲染，不因无骰而消失
- 有骰时：显示标题 "临气"、数量、总点数、来源标签（`QiDie.sourceName` 去重合并）、骰子卡片行
- 无骰时：显示斜体灰字 "暂无临时气骰" + "临时气骰可由返照、法门、状态效果等生成"
- 新增 `[详情▾]` 按钮，展开查看每枚骰子的 `性D阶(值)` + 来源追踪
- 临气骰保持虚线边框样式（`temp` CSS class）

**涉及文件**：`src/ui/combat/dice/TemporaryQiPool.tsx`（重写）、`src/styles/qi-dice.css`（新增 `.temp-qi-zone` 等样式）

---

## 二、息库 (RestPool) — 从"紧凑统计条"升级为"独立区块"

**改动前**：息库只是统计条中的一行小字 `"息库 4"`，与锁气、气池挤在一起。

**改动后**：
- 息库成为独立区块，有完整的圆角边框、header、count、total points
- 骰子以 chip 预览显示（格式：`阴D6(4)` `阳D8(5)`），一目了然
- 空态时显示 "暂无待回骰子 · 使用调息/返照回收已用骰子"
- `[详情▾]` 展开后显示每枚骰子的完整信息（性/阶/值 + 来源）
- 锁气/气池作为次要信息行显示在下方分隔线后
- 锁气骰子标记为 `locked-die`，有虚线边框

**涉及文件**：`src/ui/combat/dice/RestPool.tsx`（重写）、`src/styles/qi-dice.css`（新增 `.rest-pool-zone` 等样式）

---

## 三、资源区完整结构 (QiDiceDock)

```
┌─────────────────────────────────────────┐
│ 临气区 (TemporaryQiPool)                 │  ← 临时来源骰，始终可见
│   来源：返照 / 状态效果 / 法门            │
├─────────────────────────────────────────┤
│ ┌── 气海 (QiPool) ──┬─ 阴阳槽 ─────────┤  ← 当前宣言操作区
│ │  可用气骰，拖拽→  │  阴槽 │ 阳槽      │
│ └───────────────────┴──────────────────┘│
├─────────────────────────────────────────┤
│ 息库 (RestPool)                          │  ← 已用待回骰子
│   锁气 X 枚 · X 点 │ 气池 X 枚           │
├─────────────────────────────────────────┤
│ [确认宣言并锁气]                          │
└─────────────────────────────────────────┘
```

资源流转：气海 → 阴阳槽 → 锁气/结算 → 息库；临气 → 阴阳槽 / 特殊使用

**涉及文件**：`src/ui/combat/dice/QiDiceDock.tsx`（布局调整和注释更新）

## 息库区显示细节

- **标题**：息库（12px 粗体）
- **数量**：N 枚（有骰时 #f7e7bb 加亮）
- **总点数**：· N 点
- **骰子预览**：chip 形式，每个 chip 显示 `性D阶(值)` 如 `阴D6(4)`
- **空态**：斜体灰字 "暂无待回骰子"
- **详情展开**：列表显示每枚骰子的 `性D阶(值)` + 来源名称

## 临气区显示细节

- **标题**：临气（12px 粗体）
- **数量**：N 枚（有骰时 #f7e7bb 加亮）
- **总点数**：· N 点
- **来源**：从 QiDie.sourceName 聚合，显示如 "来源：返照 / 状态效果"
- **空态**：斜体灰字 "暂无临时气骰" + 提示文字
- **详情展开**：每枚骰子的详细信息 + 来源追踪
- **临气标记**：虚线边框 + "临" tag
