## 1. 本轮结论先行

20分钟教学可以开跑短团，但不能把所有术语一次讲完。建议采用“先教会玩家做动作，再逐步解释术语”的顺序：

1. 你想做的事是否要被规则记录？这决定是否正式出手。
2. 正式出手至少阴1阳1。
3. 出手后按流程走：宣招、截击、配置、成招、应招、落果。
4. 起手铺前置，承式读前置，绝式赌高风险。
5. 前置有三类：自身、目标、场景。
6. 势弱是喘息，势溃是清算。
7. 内功占窍不是禁用六根，只是承载与风险。
8. 非战斗也按同一流程，只是落果换成解密、危机、态度、证据。

## 2. 20分钟新手教学顺序

### 0-3分钟：告诉玩家这不是“命中检定”游戏

DM说明：

```text
这套规则关心的不是你有没有打中，而是你如何用阴阳、六根和势把局面做出来。你可以先铺局，再兑现。失败也不一定是什么都没发生，而是留下代价、破口或危机。
```

只教三个关键词：

- 阴：藏、引、化、守。
- 阳：发、冲、照、破。
- 势：当前局面的气机方向。

暂不解释全部六根细节，避免术语过载。

### 3-6分钟：正式出手边界

DM说明：

```text
如果你只是描述、看一眼、走近、问一句，通常不耗资源。
如果你想让这个动作建立前置、改变危机、改变对方态度、造成伤害、治疗、打断或被后续读取，那就是正式出手。
正式出手至少阴1阳1。
```

桌面例句：

- “我站到门边。”不算正式出手。
- “我站到门边，建立退路被看住1。”算正式出手。
- “我翻账本看看。”不算正式出手或只给叙事信息。
- “我建立账目破绽1，后面逼问掌柜。”算正式出手。

### 6-9分钟：交锋流程

只教固定口令：

```text
宣招 -> 截击窗口 -> 配置阴阳/气骰 -> 成招 -> 应招窗口 -> 落果 -> 效阶/势面更新
```

一句话区别：

- 截击：动作还没成形，去打断、干扰、降阶、改向。
- 应招：动作已经成招，落果前去抵消、转移、护人、降阶。

DM必须每轮口头提示：

```text
现在是截击窗口。
现在配置阴阳和气骰。
现在是应招窗口。
现在落果。
```

### 9-12分钟：起手、承式、绝式

DM说明：

```text
起手不是弱攻击，它负责铺局。
承式读取前置，把铺好的局兑现。
绝式需要高前置，高收益，也有高风险。
连招前置不给免费出手，只给资格或增强。
```

桌面例句：

- 起手：“我建立脚下失衡1。”
- 承式：“我读取脚下失衡1，把他逼到水边。”
- 绝式：“我读取脚下失衡、兵器被缠、水边退路，开覆舟一掷。”

### 12-15分钟：前置三色

DM说明：

```text
建立前置时必须说清放哪里：自身、目标、场景。
自身前置通常只有你能读。
目标前置打这个目标的人能读。
场景前置大家可能都能读，包括敌人和动景。
读取后默认降1层。
```

桌面追问：

```text
这个前置放你身上、目标身上，还是场景上？
谁能读？
读完是否消耗？
```

### 15-17分钟：势弱、势溃

DM说明：

```text
势弱不是失败，是大家节奏被打散，适合调息、换位、撤退、重建起手。
势溃是局面崩了，要立即清算后果。清算完会落回一个还能继续玩的势面。
你也可以主动撤势，把高张力局面退到势弱，避免势溃。
```

只给两个触发：

- 势弱：多次截击/应招、前置被清空、只拿低阶结果。
- 势溃：绝式负三、动景危机满、内功连续失稳、主动透支失败。

### 17-19分钟：内功占窍

DM说明：

```text
内功占窍不是锁死六根。你仍可用那一根。
但如果同一窍位持续承压，或被敌人针对，会先记窍压，再内功失稳，严重才六根损伤。
```

只教一条阶梯：

```text
正常 -> 窍压 -> 内功失稳 -> 六根损伤 -> 散功/留痕
```

### 19-20分钟：非战斗也同一套

DM说明：

```text
问诊、审讯、追踪、朝堂奏对也按同一流程。只是落果不是伤害，而是解密、危机、态度、证据。
```

## 3. 一页速查草案

### 行动判断

| 你想做什么 | 是否正式出手 |
|---|---|
| 只是描述、移动、问一句、看一眼 | 否 |
| 建立前置、改势、改变危机、改变态度、治疗、伤害、截击、应招、解密推进 | 是 |
| 想让刚才动作被后续机械读取 | 是 |

正式出手：至少阴1阳1。

### 流程口令

```text
宣招 -> 截击 -> 配置 -> 成招 -> 应招 -> 落果 -> 更新
```

- 截击：宣招后、配置前，处理动作能否成形。
- 应招：成招后、落果前，处理结果如何落下。
- 反击：必须写在应招条目里，不能临时获得。
- 同一窗口默认最多两名响应者，其余可援助或保留。

### 前置三色

| 类型 | 谁能读 | 读后 |
|---|---|---|
| 自身 | 建立者 | 默认降1层 |
| 目标 | 针对该目标者 | 默认降1层 |
| 场景 | 场内符合条件者，含敌人/动景 | 通常降1层；地形常态可保留但推危机 |

建立前置时问：放哪里？谁能读？读后是否消耗？

### 势面

| 势面 | 桌面含义 |
|---|---|
| 阴盛 | 藏、牵、诱、化、暗线有利 |
| 阳盛 | 冲、压、破、威慑有利 |
| 合势 | 多人前置互相接上 |
| 圆融 | 高稳定、高整合，通常需多重前置 |
| 势弱 | 节奏打散，适合调息、换位、撤退、重建起手 |
| 势溃 | 结构崩坏，立即清算后落回可玩势面 |

### 占窍阶梯

```text
正常 -> 窍压 -> 内功失稳 -> 六根损伤 -> 散功/留痕
```

- 占窍不禁用六根。
- 同窍持续承压才记窍压。
- 普通失败通常只到窍压。
- 负二以上、连续针对、绝式负三才推进更深。

### 非战斗落果

| 类型 | 正效阶 | 负效阶 |
|---|---|---|
| 解密 | 线索、确认、许可 | 误判、假线索 |
| 危机 | 危机-1、延缓动景 | 危机+1、动景推进 |
| 态度 | 改善、迟疑、配合 | 恶化、设防、反问 |
| 证据 | 稳固、转交、公开 | 污染、暴露、反向前置 |

## 5. 复测结果

20分钟教学足够开一场短团，但需要DM严格控制范围：

- 不要在开局讲所有六根细节。
- 不要先讲完整数据库字段。
- 不要一次讲所有势面特殊用法。
- 不要让玩家先设计复杂招式包。

最适合新手第一场的角色卡应该只包含：

- 两个擅长六根，一个短板六根。
- 一个内功占窍。
- 一个起手，一个承式，一个应招，一个绝式。
- 一格阴槽/阳槽记录。
- 当前前置、当前势面、当前占窍状态。

## 6. 对规则书的直接补丁建议

### 6.1 新手入口建议稿

```text
第一次开团时，不需要先阅读全部规则。玩家只需理解：正式出手至少阴阳各1；截击在宣招后配置前，应招在成招后落果前；起手建立前置，承式读取前置，绝式承担高风险；前置分自身、目标、场景；势弱可重整，势溃要清算；内功占窍不是禁用六根。
```

### 6.2 DM屏内容建议

DM屏必须放：

- 正式出手三分法。
- 交锋流程口令。
- 截击/应招区别。
- 前置三色。
- 势弱/势溃边界。
- 占窍阶梯。
- 非战斗落果四类。

### 6.3 第一场短团建议

第一场短团应至少安排：

1. 一个自由叙事动作和一个正式出手动作，用来教边界。
2. 一个敌方宣招，让玩家截击。
3. 一个敌方成招，让玩家应招。
4. 一个目标前置和一个场景前置。
5. 一次势弱重整。
6. 一个非战斗交锋，例如问诊、追踪或审讯。
