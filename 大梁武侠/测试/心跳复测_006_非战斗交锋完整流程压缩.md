## 1. 本轮裁定假设

非战斗交锋不另开小系统，仍使用：

```text
宣招 -> 截击窗口 -> 配置阴阳/气骰 -> 成招 -> 应招窗口 -> 落果 -> 正负效阶 -> 势面/前置/危机更新
```

但落果类型从“伤害/状态”换成以下四类：

| 落果类型 | 正效阶示例 | 负效阶示例 |
|---|---|---|
| 解密 | 解密+1、获得线索、确认前置 | 误判、假线索、危机+1 |
| 态度 | 态度改善、让对方迟疑、打开谈判许可 | 态度恶化、对方设防、被反问 |
| 证据 | 证据稳固、可转交、可公开 | 证据污染、暴露来源、反向前置 |
| 危机 | 危机-1、延缓动景、清除风险 | 危机+1、动景推进、势弱/势溃 |

非战斗交锋是否需要“敌方/动景响应”取决于对象：

- 死物/普通静景：通常不可响应，只按效阶推进。
- 人物：可截击/应招，尤其在审讯、谈判、朝堂奏对中。
- 动景：可响应，也可被响应，例如瘟疫扩散、巡夜查院、朝堂攻讦。
- 证据/场景前置：可能被反向读取。

## 4. 复测结果

### 4.1 非战斗交锋可以共用完整流程

四类场景都能接入同一流程：

- 问诊：病人不响应，病症动景响应。
- 追踪：死物/静景不响应，直接按效阶推进。
- 审讯：人物可以截击话术，也可以应招转移证词。
- 朝堂奏对：攻讦可作为动景，证据可作为场景前置。

这说明规则书不需要另设“技能挑战”，但需要给不同对象类型的响应边界。

### 4.2 非战斗落果要有固定词汇

建议非战斗落果统一使用四组词：

- 解密：+1/+2/+3，代表线索推进。
- 危机：+1/-1，代表场景压力。
- 态度：敌对、设防、迟疑、中立、配合。
- 证据：线索、证词、物证、公开证据、反向前置。

这样DM不用每次临场发明结果。

### 4.3 人物响应让社交不再是单向检定

掌柜截击、主簿应招让非战斗交锋有对抗感：

- 截击可以关闭玩家铺局资格。
- 应招可以降阶、转移目标、附带代价。
- 但不应频繁完全取消玩家成果，否则社交会挫败。

### 4.4 非战斗更容易触发“正式出手边界”争议

玩家在非战斗中更常说“我只是站门口”“我只是问一句”。复测001的三分法仍有效：

- 只是描述，不算正式出手。
- 若要建立前置或改变态度/危机/证据，算正式出手。

### 4.5 势面在非战斗中仍有用

势弱在朝堂中表示“无人完全压住局面”，适合谈判、守势、保留余地。阳盛可对应压场、攻讦、威慑；阴盛可对应藏证、诱导、暗线推进；合势可对应多方证据互相印证。

## 5. 对规则书的直接补丁建议

### 5.1 非战斗对象响应建议稿

```text
非战斗交锋仍使用交锋轮。对象为死物或普通静景时，通常不响应，只按正负效阶推进。对象为人物、群体或动景时，可按条目截击或应招。病症、火势、巡夜、朝堂攻讦、瘟疫扩散等均可视为动景。
```

### 5.2 非战斗落果建议稿

```text
非战斗交锋的正负效阶可落到解密、危机、态度、证据四类结果。正效阶推进线索、降低危机、改善态度或稳固证据；负效阶造成误判、危机增加、态度恶化、证据污染或产生反向前置。
```

### 5.3 人物应招建议稿

```text
社交、审讯、朝堂奏对中的人物应招，不宜频繁完全取消玩家成果。更常见的应招结果是降阶、转移证词对象、附加政治代价、改变态度幅度、或生成反向前置。
```
