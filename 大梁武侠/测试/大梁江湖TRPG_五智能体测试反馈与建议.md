## 1. 记录官访谈

### 1.1 玩家A：沈却潮

记录官：你用一句话形容这次上桌体验？

玩家A：像是在打“气机”和“局面”，不是单纯换血。水网角色能通过脚下、水声、桥索这些东西发挥作用。

记录官：你最喜欢的机制是什么？

玩家A：起手和承式。起手不是弱攻击，而是铺局。等我用折波缠腕读取前置时，会觉得前一轮没白做。

记录官：最困惑的地方？

玩家A：正式出手的边界。观察、移动、试探、铺前置，有些像叙事，有些像规则动作。如果都要阴阳各1，新手会怕做小动作。

记录官：内功占窍清楚吗？

玩家A：概念很好，但需要例子。我一开始以为丹田被潮息诀占了，就不能再用丹田。后来DM裁定“能用，但被针对时先伤内功稳定”，这就清楚很多。

记录官：截击和应招能分清吗？

玩家A：跑过一次后能分清。截击是动作还没成形前去断，应招是结果要落下来前处理。但桌上必须每轮提醒窗口。

记录官：继续长团最想保留什么？

玩家A：势和前置。还有地区武学差异。东南水网不是背景皮肤，真的会改变我怎么出手。

### 1.2 玩家B：岑雾娘

记录官：你的医毒/社交角色是否有存在感？

玩家B：有。因为非战斗也能用起手、承式、效阶。问诊、账本、治疗不是“我过一个技能检定”，而是能建立前置给队友读。

记录官：最大疑惑是什么？

玩家B：非战斗行动是否也进入阴阳槽。医毒、审问、潜行、探查如果算正式出手，阴阳各1会很有味道，但规则书要给例子。

记录官：医疗绝式的失败后果是否合理？

玩家B：合理。医疗绝式失败不该只是我掉血，而是误诊、毒走命门、病人欠假气。这个方向很有戏。

记录官：哪个规则最像负担？

玩家B：内功占窍如果没有清楚的持续时间、冲突和解除方式，会变成查询负担。它必须是选择，不是记忆惩罚。

记录官：你认为最有武侠味的机制？

玩家B：正负效阶和前置。能把“看出病根”“逼出破绽”“救人留后患”都写进机制。

### 1.3 玩家C：祁照野

记录官：祁照野作为武斗/轻功角色，是否会变成纯输出？

玩家C：不会。他更像把局面撞开的人。抢位、断后、追击、护人都能成为正式行动。

记录官：阴阳各1是否限制表达？

玩家C：反而帮我防止无脑堆阳。阴槽像保命和变向，阳槽像胆气和爆发。

记录官：最大规则问题？

玩家C：前置归属。轻功角色常建立“高处”“侧后”“狭檐”这种地形前置，要明确队友能不能读。

记录官：负效阶体验如何？

玩家C：如果只是扣资源会无聊。但西北军镇那场，我负一阶暴露身份，换来阳盛起势，这很好。失败也在推进故事。

记录官：绝式风险如何？

玩家C：可以高风险，但宣招前要知道大概会亏什么。势溃、被反制、内功受损、位置失控都能接受。

### 1.4 DM：Cicero

记录官：最难主持的部分是什么？

DM：同时盯前置、势面、阴阳槽、内功占窍和响应窗口。规则目标我理解，但桌面需要工具。

记录官：玩家最常卡在哪里？

DM：三个地方：正式出手边界、截击/应招时点、内功占窍是不是等于使用六根。

记录官：起手、承式、绝式是否帮助组织战斗？

DM：帮助很大。它把武侠战斗从“我打你一下”变成“试探、铺局、破招、爆发”。但条目必须写清楚承式读取什么前置。

记录官：势面是否好用？

DM：好用，但势弱和势溃边界需要更明确。阴盛、阳盛、合势、圆融比较容易讲，势弱和势溃更依赖临场裁定。

记录官：最需要规则书补例子的地方？

DM：一页流程速查、正式出手例子、非战斗交锋例子、前置传递、绝式失败表、内功占窍冲突。

## 2. 主要发现

### 2.1 成立的设计方向

1. “规则即世界观”成立。  
   玩家能把东南水网、西北军镇、南疆医毒、上京权术转化为不同打法，而不是只当背景。

2. 起手、承式、绝式形成武侠节奏。  
   起手铺局，承式读取，绝式冒险，能自然产生试探、蓄势、破招和反噬。

3. 非战斗交锋可用同一套底座。  
   问诊、账本、朝堂奏对、断桥动景都能按阴阳、六根、前置、效阶处理。

4. 负效阶可以推动故事。  
   玩家接受暴露身份、误诊、假气续命、桥索半断等非扣血后果。

5. 内功占窍有潜力。  
   当它同时提供稳定能力和可被针对的弱点时，玩家认为有取舍。

### 2.2 高风险问题

1. 正式出手边界不清。  
   玩家反复问普通观察、移动、问诊、站位、喊话什么时候算正式出手。

2. 截击和应招容易混。  
   两者时点有特色，但新玩家需要DM每轮提示：“现在是截击窗口”“现在是应招窗口”。

3. 内功占窍容易误解。  
   玩家会把“使用六根”误解为“占窍”，或以为被占窍后该六根不能再用。

4. 前置归属需要规则口径。  
   地形、病理、社交、位置前置到底是个人可读、全队可读，还是建立者指定，需要写清。

5. 势弱与势溃边界不足。  
   阴盛、阳盛、合势、圆融容易理解；势弱和势溃何时触发、触发后具体后果，需要表格。

6. DM追踪压力偏高。  
   没有工具时，DM要同时追阴阳槽、气骰、前置、势面、占窍、效阶，容易漏窗口。

## 3. 机制建议

### 3.1 正式出手边界

建议在规则书增加“三分法”：

| 行动类型 | 是否正式出手 | 示例 |
|---|---|---|
| 自由叙事 | 否 | 普通说话、走近、看一眼、描述姿态 |
| 结构化铺局 | 是 | 建立前置、改势、解密、压危机、为承式开条件 |
| 直接落果 | 是 | 伤害、状态、夺势、破前置、治疗、截击、应招 |

建议例句：

```text
玩家只是说“我站到门边”，不算正式出手。
玩家说“我站到门边，建立退路被我看住1，供后续逼问读取”，算正式出手，至少投入阴阳各1。
```

### 3.2 截击/应招速查

建议每张角色卡或DM屏放固定流程：

```text
1. 宣招
2. 截击窗口：打断或干扰动作成形
3. 配置阴阳槽和气骰
4. 成招
5. 应招窗口：降低、转移、抵消、护人、反弹落果
6. 落果
7. 正负效阶兑现
8. 势面、前置、状态更新
```

一句话区别：

- 截击处理“动作能不能顺利成形”。
- 应招处理“已经成招的结果如何落下”。

### 3.3 内功占窍

建议明确四条：

1. 调用六根不等于占窍。
2. 占窍是内功长期入身，占用构筑空间。
3. 被占之窍仍可使用，但被针对时优先影响内功稳定。
4. 同一轮或同一场景超负荷调用被占之窍，可换强效果，但记反噬前置。

示例：

```text
潮息诀占丹田。沈却潮仍可用丹田调息。
若他同时用潮息诀稳桥并强行压蛊声，成功可额外稳定场景；失败则记“丹田疲滞1”。
若敌人用封丹田招式命中，优先使潮息诀失稳，而不是直接禁止所有丹田行动。
```

### 3.4 前置归属

建议分为三类：

| 前置类型 | 默认归属 | 示例 |
|---|---|---|
| 自身前置 | 仅建立者可读 | 蓄劲、藏锋、内息回环 |
| 目标前置 | 攻击该目标者可读 | 命门虚浮、脚下失衡、被逼至水边 |
| 场景前置 | 全场可读或被动景读取 | 桥索半断、遮蔽、账目破绽 |

建立前置时建议强制说清：

```text
这个前置放在我身上、目标身上，还是场景上？
谁能读取？
读取后消耗、降低，还是保留？
```

### 3.5 势弱与势溃

建议补清边界：

| 势面 | 建议定义 | 典型后果 |
|---|---|---|
| 势弱 | 节奏不稳但结构未破 | 最高正效阶受限、前置难滚大、适合调息撤退 |
| 势溃 | 行动结构崩坏 | 立即结算严重后果：内功失稳、位置崩坏、前置清空、被反制 |

建议势溃不要频繁触发，只在以下情况出现：

- 绝式负三阶。
- 关键前置被反向读取。
- 动景危机达到上限。
- 内功占窍被连续针对。
- 玩家主动透支并失败。

- 响应窗口提示牌：截击、应招。

2. 医毒问诊如何使用阴阳槽。

4. 地形前置如何给队友读取。
5. 截击失败如何处理。
6. 应招能否反击。
7. 绝式失败的负三阶。
8. 占窍被攻击后的处理。
9. d4再降骰时转为负面前置。
10. 势弱和势溃的区别。

## 6. 总结判断

本轮模拟结论：规则底座有明显可玩性，尤其适合强调武侠中的试探、借势、破招、反噬和伦理选择。它不只是战斗系统，也能承载问诊、追查、朝堂、动景等情景。

下一步不建议先扩大量新内容，而应先补“上桌可执行性”：流程速查、例子、前置归属、占窍口径、势弱/势溃表。只要这些补齐，短团教学和长团持续游玩都会稳定很多。
