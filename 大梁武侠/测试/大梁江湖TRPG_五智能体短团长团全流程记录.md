## 0. 测试说明

本记录由五个子智能体分别模拟三名玩家、一名DM、一名记录官，并由主线程整合。流程按真实桌面习惯书写：先开卡，再短团，再长团压缩跑完，最后结团访谈。

本轮重点测试：

- 阴槽、阳槽：正式出手至少阴阳各1。
- 六根：顶门、心口、目窍、步根、丹田、命门。
- 气骰：最低d4，降骰不低于d4。
- 势面：阴盛、阳盛、势弱、合势、圆融、势溃。
- 起手、承式、绝式：起手铺势和前置，承式读取前置，绝式高前置高风险。
- 响应窗口：截击在宣招后、配置前；应招在成招后、落果前。
- 内功占窍：内功提供稳定特性，同时成为可被针对的弱点。
- 情景交锋：问诊、审讯、断桥、朝堂奏对等非战斗行动也可使用交锋轮和正负效阶。

本记录中出现的掷骰样本来自 rpg-tools 的 `dice.py` 实际运行：

- `2d6+1`：掷出 `[1, 3]`，结果5。
- `1d8+1d6`：掷出 `[6]` 与 `[1]`，结果7。
- `2d4`：掷出 `[2, 1]`，结果3。
- `3d6kh2`：掷出 `[5, 3]`，结果8。
- `1d10+1d6`：掷出 `[6]` 与 `[3]`，结果9。

## 1. 参团人员

### DM：Cicero

主持目标：验证规则能否支持真实武侠桌面节奏，而不是只在条目范式上成立。

主持固定口头流程：

```text
宣招 -> 截击窗口 -> 配置阴阳槽与气骰 -> 成招 -> 应招窗口 -> 落果 -> 效阶兑现 -> 势面与前置更新
```

### 记录官：Herschel

记录原则：

- 优先保留原话。
- 每次规则疑惑标记事件标签。
- 同步记录阴阳消耗、气骰、前置、势面、截击/应招窗口、正负效阶。
- 结团后分别采访三名玩家和DM。

### 玩家A：沈却潮

出身：东南水网，船帮押货人，听潮门弟子。  
定位：偏阴控场、步根/目窍/丹田，擅长借水势、绕身位、截对手发力。  
内功：潮息诀，占丹田窍。  
短板：顶门、命门。

招式摘要：

- 起手「听浪寻缝」：建立阴盛或合势，给目标挂“脚下失衡”“身位偏斜”“呼吸被牵”。
- 承式「折波缠腕」：读取前置，压制对方兵器或步法。
- 绝式「覆舟一掷」：需要多个前置，失败可能势溃或露命门。

### 玩家B：岑雾娘

出身：南疆药寨外姓女，随商队入东南水网，曾在上京药铺做账房。  
定位：探索、医毒、法门、社交。  
六根：心口、目窍、丹田较强；命门偏弱。  
核心法门：《青瘴听脉法》《水灯迷步》《七叶回魂针》。

招式摘要：

- 起手「青瘴听脉」：问诊、辨毒、识破气机。
- 承式「七叶回针」：读取病症或破绽前置，治疗或扰乱。
- 绝式「七叶返照」：高风险医疗绝式，可反逼毒性，也可能伤及病人命门。

### 玩家C：祁照野

出身：西北军镇边驿，曾为军马队探路、送急令。  
定位：武斗、轻功、胆魄，负责抢位、追击、断后、救场。  
六根：步根最高，命门次高；心口偏低。  
内功：边军吐纳，占丹田或命门一窍。

招式摘要：

- 起手「抢步立势」：建立“高处”“侧后”“先声夺人”等前置。
- 承式「渡阵逼门」：读取地形或压迫前置，突进、逼退、护人。
- 绝式「坠步斩关」：从高位或侧翼切入，失败接受势溃或反制。

### 3.4 短团结局

结局：正负相抵偏正。药匣保住，桥局部损毁，沈浮灯逃走但留下上京线索。  
获得长团钩子：

- 上京有人买凶也有人续命。
- 药匣是活蛊封印，用于压制命门旧伤。
- 沈却潮的师兄之死可能与“断潮客”有关。
- 岑雾娘发现南疆禁药链条。
- 祁照野的失落军令与西北禁法有关。

短团即时反馈：

## 4. 长团《六根入梁》压缩记录

长团以五回压缩跑完，每回选一个关键场景实跑。目标不是展开全部剧情，而是测试规则在长期结构中是否会固化、是否能支持角色成长。

### 4.1 第一回：东南水网，失匣

核心问题：水网帮派靠漕运、采莲、夜渡谋生，武学偏阴，情景交锋能否和武学交锋同构。

[长团01-00:40] 玩家A：那我改用目窍和步根，不强压丹田。
[长团01-01:00] 记录官：[长团成长] 玩家开始因上一场代价调整打法。

[长团01-03:10] 玩家C：我负责追影子，建立“船舷侧后1”。

## 5. 结团记录

### 5.1 角色归宿

沈却潮：

- 没有杀断潮客。
- 得知师兄之死与续命法债链有关。
- 潮息诀的丹田疲滞在长团中逐步恢复。
- 未来目标：重整听潮门水路规矩，防止水网武学被黑市禁法绑架。

岑雾娘：

- 建成第一版跨地区医案簿。
- 证明医毒角色不只是“毒伤输出”，而是能推动解密、救治、社交与终局伦理选择。
- 青囊护息在结团后可散功，留下“首次问诊可确认一项六根损伤”的洗髓留痕。

祁照野：

- 查明边驿旧案与命门禁法有关。
- 绝式高光不是杀人，而是斩断军令木牌与守住观星台门。
- 未来武学雏形：“照野渡阵步”，核心是人在绝境里还有一步可走。

### 5.2 机制高光

- 短团中三人分别稳桥、逼位、控蛊，形成圆融前置。
- 祁照野在西北军镇用负效阶暴露身份，换来阳盛起势，负效阶产生叙事推进。
- 岑雾娘在上京用账本和医案打社交承式，非战斗交锋成立。
- 沈却潮面对断潮客时选择先看六根债，不直接复仇，规则线索改变角色选择。

## 6. 结论

本次模拟实跑显示，该系统的核心吸引力来自“先铺势，再兑现”的武侠节奏，以及地区生计、六根、阴阳、前置之间的连动。短团能跑出规则味，长团能跑出角色成长与主题选择。

主要风险集中在学习与主持成本：正式出手边界、内功占窍、截击/应招窗口、前置归属、势弱/势溃边界，都需要规则书给更多例子和一页桌面速查。
