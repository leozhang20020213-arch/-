## 一、新手开团页

### 1. 这套规则在玩什么

《大梁江湖TRPG》不是先问“打没打中”，而是先问：你如何调动阴阳、六根、前置与势，把局面做出来。

你可以先试探、铺局、牵制、逼位、辨病、诱供、压场，再用承式或绝式兑现。失败也不一定是什么都没发生，常见结果是留下破口、失位、危机、反向前置或内功失稳。

第一次开团不需要读完整规则。先记住八件事：

1. 如果行动要被规则记录，或要被后续读取，就是正式出手。
2. 正式出手至少投入阴1阳1。
3. 出手流程固定：宣招、截击、配置、成招、应招、落果、更新。
4. 起手负责铺局，承式负责读取前置，绝式负责高风险兑现。
5. 前置分三类：自身、目标、场景。
6. 势弱是节奏打散，可以重整；势溃是结构崩坏，要立即清算。
7. 内功占窍不是禁用六根，而是承载能力与风险。
8. 非战斗也用同一流程，只是落果换成解密、危机、态度、证据。

### 2. 阴阳怎么理解

阴不是邪，阳不是正。阴阳只表示气机方向。

| 阴 | 阳 |
|---|---|
| 藏、引、化、守、听、避 | 发、冲、照、破、压、断 |
| 卸力、潜行、识破、反照、医毒、机关 | 强攻、震慑、破阵、追击、压场 |

正式出手至少阴阳各1。你可以偏阴，也可以偏阳，但不能用纯阴或纯阳完成完整武学行动。

例：

- 沈却潮用阴听水、用阳拨桥索，建立“看破水线1”。
- 岑雾娘用阴辨药、用阳落笔成证，建立“账目破绽1”。
- 祁照野用阳抢步、用阴卸力，建立“高位压迫1”。

### 3. 什么算正式出手

判断标准：

| 行动 | 是否正式出手 |
|---|---|
| 只是走近、看一眼、问一句、描述姿态 | 否 |
| 建立前置 | 是 |
| 改变势面、危机、态度、证据、状态 | 是 |
| 造成伤害、治疗、截击、应招、解密推进 | 是 |
| 想让刚才动作被后续机械读取 | 是 |

同一句描述可以因意图不同而改变分类：

```text
“我站到门边。”不算正式出手。
“我站到门边，建立退路被看住1。”算正式出手。
```

```text
“我翻账本看看。”可以只获得叙事信息。
“我翻账本，建立账目破绽1，后面逼问掌柜。”算正式出手。
```

### 4. 交锋流程

每次正式交锋按这个顺序走：

```text
宣招 -> 截击窗口 -> 配置阴阳/气骰 -> 成招 -> 应招窗口 -> 落果 -> 效阶/势面/前置更新
```

一句话记忆：

- 截击：动作还没成形，去打断、干扰、降阶、改向。
- 应招：动作已经成招，结果落下前去抵消、转移、护人、降阶。

应招中的反击必须由条目写明，不能从通用防守成功临时获得。

同一响应窗口默认最多结算两名响应者。其他人可以援助或保留到下一窗口。

### 5. 起手、承式、绝式

起手不是弱攻击，承式不是免费第二刀，绝式不是无脑大招。

| 类型 | 作用 |
|---|---|
| 起手 | 建立前置、改变势面、打开承式资格 |
| 承式 | 读取前置，扩大优势，兑现中等收益 |
| 绝式 | 读取多重前置，高收益，高风险 |

示例：

```text
起手：我建立“脚下失衡1”。
承式：我读取“脚下失衡1”，把他逼到水边。
绝式：我读取“脚下失衡”“兵器被缠”“水边退路”，开覆舟一掷。
```

连招前置只给资格或增强，不赠送额外出手。

### 6. 前置三色

建立前置时，必须说清它放在哪里。

| 前置类型 | 谁能读 | 例子 | 读后 |
|---|---|---|---|
| 自身前置 | 建立者本人 | 蓄劲、藏锋、屏息、内息回环 | 默认降1层 |
| 目标前置 | 针对该目标行动者 | 脚下失衡、命门虚浮、被逼至水边 | 默认降1层 |
| 场景前置 | 场内符合条件者，包括敌人和动景 | 雨棚低梁、账目破绽、桥索半断 | 通常降1层；地形常态可保留但推动危机 |

DM追问三句：

```text
这个前置放你身上、目标身上，还是场景上？
谁能读取？
读取后是否消耗？
```

场景前置强，但也危险，因为敌人和动景可能反向读取。

### 7. 势弱与势溃

| 势面 | 桌面含义 |
|---|---|
| 阴盛 | 藏、牵、诱、化、暗线有利 |
| 阳盛 | 冲、压、破、威慑有利 |
| 合势 | 多人前置互相接上 |
| 圆融 | 高稳定、高整合，通常需要多重前置 |
| 势弱 | 节奏被打散，适合调息、换位、撤退、谈判、重建起手 |
| 势溃 | 结构崩坏，立即清算后果，清算后落回可继续游玩的势面 |

势弱不是失败。它是喘息、重整、换打法的窗口。

势溃不是普通劣势。它是局面崩坏点，常见于绝式负三、动景危机满、内功连续失稳、主动透支失败。

玩家可以主动撤势，把高张力局面退到势弱，以避免继续滚向势溃。

### 8. 内功占窍

占窍表示内功长期承载于某一六根窍位，不表示该六根不能再用。

你仍然可以调用被占的六根出招、应招、问诊、观察或行法。只是当同一窍位持续承压，或被敌人针对时，先记录风险。

占窍阶梯：

```text
正常 -> 窍压 -> 内功失稳 -> 六根损伤 -> 散功/留痕
```

普通失败通常只到窍压。负二以上、连续针对、绝式负三，才推进到更深层级。

### 9. 非战斗交锋

问诊、追踪、审讯、朝堂奏对也按同一交锋流程走。

对象不同，响应不同：

- 死物/普通静景：通常不响应，只按效阶推进。
- 人物/群体：可截击或应招。
- 动景：可响应，也可被响应。例如病症、火势、巡夜、朝堂攻讦。

非战斗落果分四类：

| 类型 | 正效阶 | 负效阶 |
|---|---|---|
| 解密 | 线索、确认、许可 | 误判、假线索 |
| 危机 | 危机-1、延缓动景 | 危机+1、动景推进 |
| 态度 | 改善、迟疑、配合 | 恶化、设防、反问 |
| 证据 | 稳固、转交、公开 | 污染、暴露、反向前置 |

## 二、20分钟教学流程

### 0-3分钟：规则目标

只讲一句：

```text
这套规则玩的是试探、铺局、破招、反噬和兑现，不是单纯换血。
```

只教阴、阳、势三个词。

### 3-6分钟：正式出手

教会玩家判断：

- 只是叙事，不耗。
- 要被记录、读取、改变局面，就正式出手。
- 正式出手至少阴1阳1。

### 6-9分钟：流程口令

让玩家跟读：

```text
宣招、截击、配置、成招、应招、落果、更新。
```

### 9-12分钟：起手/承式/绝式

用一个例子讲完：

```text
起手建立脚下失衡。
承式读取脚下失衡逼到水边。
绝式读取多个前置赌高风险。
```

### 12-15分钟：前置三色

让玩家每次建前置都回答：

```text
放自身、目标，还是场景？
```

### 15-17分钟：势弱/势溃

只讲：

- 势弱：能重整。
- 势溃：要清算。

### 17-19分钟：占窍

只讲：

```text
占窍不是禁用六根。先窍压，再失稳，再损伤。
```

### 19-20分钟：非战斗也同构

只讲：

```text
问诊、审讯、追踪、朝堂奏对也用同一流程，落果换成解密、危机、态度、证据。
```

## 三、DM屏速查

### 行动判断

```text
正式出手 = 要被规则记录 / 被后续读取 / 改变状态、危机、势面、前置、资源、效阶。
正式出手至少阴1阳1。
```

### 流程

```text
宣招 -> 截击 -> 配置 -> 成招 -> 应招 -> 落果 -> 更新
```

截击：宣招后配置前。  
应招：成招后落果前。  
反击：必须条目明写。  
同一窗口：默认最多两名响应者。

### 前置

```text
自身：本人读。
目标：针对目标者读。
场景：符合条件者读，敌人/动景也可读。
默认读取后降1层。
```

### 势面

```text
阴盛：藏、牵、诱、化。
阳盛：冲、压、破、威慑。
合势：多人接上。
圆融：高稳定整合。
势弱：节奏打散，可重整。
势溃：结构崩坏，立即清算。
```

### 占窍

```text
占窍不禁用六根。
正常 -> 窍压 -> 内功失稳 -> 六根损伤 -> 散功/留痕。
```

### 非战斗落果

```text
解密：线索/误判。
危机：降低/增加。
态度：改善/恶化。
证据：稳固/污染/反向前置。
```

## 四、第一场短团教学清单

第一场短团必须安排六个教学点：

1. 一个自由叙事动作和一个正式出手动作，教正式出手边界。
2. 一个敌方宣招，让玩家截击。
3. 一个敌方成招，让玩家应招。
4. 一个目标前置和一个场景前置。
5. 一次势弱重整。
6. 一个非战斗交锋，例如问诊、追踪或审讯。

建议第一场不要安排：

- 多段复杂连招。
- 多个内功同时占窍。
- 超过三层的前置。
- 大规模多人响应。
- 多个动景同时运转。
