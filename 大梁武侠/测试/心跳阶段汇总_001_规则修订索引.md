## 2. 稳定结论

### 2.1 正式出手三分法稳定

可直接写入规则书：

```text
当一个行动要被规则记录、被后续读取，或直接改变人物/场景的状态、危机、势面、前置、资源、效阶时，称为正式出手。正式出手至少投入阴、阳各1。

普通说话、移动、观看、触摸、描述姿态等，如果只用于获得叙事信息，不建立前置、不改变势面、不直接落果，可作为自由叙事处理，不消耗阴阳槽。

同一叙事动作可以因玩家意图不同而改变分类。例如“站到门边”若只是叙事位置，不算正式出手；若声明建立“退路被看住1”，则算正式出手。
```

建议放置位置：新手入口、行动章、情景交锋章。

### 2.2 响应窗口口径稳定

可直接写入规则书：

```text
截击发生在宣招后、配置前，处理动作能否成形；应招发生在成招后、落果前，处理结果如何落下。

截击失败时，原动作通常继续进入配置或成招流程。截击者不默认承受原动作落果，除非截击者本来就是该动作目标，或截击条目写明失败时卷入落果。

应招可以降低、抵消、转移、护人、脱离或反弹已经成招的落果。反击不是通用应招结果。只有应招条目明确写有反击，并满足其效阶、成本、距离、武器或前置条件时，才可在应招中反击。
```

建议放置位置：响应章、截击/应招范式、DM屏。

### 2.3 前置三色稳定

可直接写入规则书：

```text
建立前置时，必须标明前置归属：自身、目标或场景。

自身前置只供建立者读取，例如蓄劲、屏息、藏锋、内息回环。
目标前置附着在某个对象上，针对该对象行动者可读取，例如脚下失衡、命门虚浮、被逼至水边。
场景前置附着在环境、证据、局势或动景上，场内符合条件者都可能读取，包括敌人和动景。

读取前置时，声明读取哪个前置、由谁读取、用于哪个行动。除条目另有说明外，前置被读取后降低1层；降至0时清除。同一前置同一轮最多被读取两次。
```

建议放置位置：起手/承式章、连招前置章、情景交锋章。

### 2.4 非战斗交锋同构稳定

可直接写入规则书：

```text
非战斗交锋仍使用交锋轮。对象为死物或普通静景时，通常不响应，只按正负效阶推进。对象为人物、群体或动景时，可按条目截击或应招。病症、火势、巡夜、朝堂攻讦、瘟疫扩散等均可视为动景。

非战斗交锋的正负效阶可落到解密、危机、态度、证据四类结果。正效阶推进线索、降低危机、改善态度或稳固证据；负效阶造成误判、危机增加、态度恶化、证据污染或产生反向前置。
```

建议放置位置：法门章、情景交锋章、DM指南。

## 3. 中高稳定但仍需一次实桌复验

### 3.1 内功占窍阶梯

当前建议稿：

```text
占窍表示内功长期承载于某一六根窍位，并不禁止角色继续调用该六根。角色仍可用该六根出招、应招、问诊、观察或行法。但若同一场景内反复以被占窍位承压，或被敌方针对该窍位，先记录“窍压”或“内功失稳”，再视严重程度进入六根损伤。

当被占窍位受到针对时，按以下阶梯处理：
1. 窍压：内功仍可用，但该窍位成为后续前置。
2. 内功失稳：内功效果受限、调息变难、相关效阶上限降低或风险增加。
3. 六根损伤：该六根本体受创，影响更广泛的行动。
4. 散功：内功关闭，按条目判断是否留下洗髓留痕。
```

仍需验证：

- 这条阶梯是否会造成状态记录过多。
- “窍压”和“内功失稳”的差别是否足够明显。
- 不同内功是否需要自带更短的专属受击表。

### 3.2 势弱/势溃边界

当前建议稿：

```text
势弱表示交锋节奏被打散，但局面尚未崩坏。当前势弱时，角色难以直接读取高阶连招或绝式前置，但适合调息、换位、撤退、清除前置、降低危机、重建起手或谈判。

势溃表示某一方或场景结构崩坏，必须立即清算后果。势溃不是普通劣势，也不是可长期维持的势面池。势溃清算后，DM应将场面落回一个可继续游玩的势面，如势弱、阴盛或阳盛。
```

仍需验证：

- 玩家是否会故意频繁撤到势弱，导致绝式风险被过度规避。
- 势溃清算表是否需要按人物、动景、内功、场景分类型。
