# 大梁江湖TRPG APP 投放图标资源包（2026年6月20日）

本包从本轮对话生成的图标中挑选了适合正式投放的一批，剔除了概念展示图、带多图排版且不适合直接使用的图、过度复杂或风格不统一的图。

## 目录说明

- `assets/icons/png512`：大卡、封面、高清预览使用。
- `assets/icons/png256`：推荐前端默认使用尺寸，清晰且体积适中。
- `assets/icons/png128`：小卡、列表、状态面板使用。
- `assets/icons/webp512`：Web / Electron / 小程序优先使用，体积更小。
- `assets/icons/source_full`：原图或统一处理后的源图，便于后期二次裁切。
- `assets/icons/content_crops`：从白底图标页裁切出的原始白底小图，仅供回溯。
- `docs/icon_manifest.json`：开发读取用完整投放表。
- `docs/icon_manifest.csv`：人工核对用表格。
- `docs/daliangIconMap.ts`：前端路径映射示例。
- `preview/selected_icons_contactsheet.png`：已选图标总览。
- `reference`：候选图总览和未拆分原图，仅作参考，不建议直接投放。

## 推荐使用口径

1. APP正式投放优先用 `png256` 或 `webp512`。
2. 底部导航、小按钮、状态角标用 `png128`。
3. 角色页、交锋页、武学详情页的大卡片用 `png512`。
4. 规则口径中势面采用：阴盛、阳盛、合势、圆融、崩溃、失势，不使用“夺势”。
5. 六根/内功盘按当前修正口径：头部/顶门、眼、心、丹田、腰、腿。

## Codex投放建议

让 Codex 将 `assets/icons` 拷贝到项目静态资源目录，例如：

- React / Vite：`src/assets/icons` 或 `public/assets/icons`
- Flutter：`assets/icons`，并在 `pubspec.yaml` 中登记
- Electron：`resources/assets/icons` 或前端 public 目录

然后读取 `docs/icon_manifest.json` 或复制 `docs/daliangIconMap.ts` 建立路径映射。
