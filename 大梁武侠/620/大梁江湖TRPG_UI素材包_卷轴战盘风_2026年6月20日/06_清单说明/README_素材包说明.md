# 大梁江湖TRPG UI素材包（卷轴战盘风）

本包根据当前确认的“卷轴战盘风”UI方向，将已生成的资源板拆分为更适合前端接入的分类素材。

## 目录说明
- 00_整板总览：保留原始整板，便于回看整体风格。
- 01_全局框架与导航：启动页、房间、导航、侧栏、弹窗、底部栏。
- 02_交锋与裁定：交锋圆盘、敌我卡、DM裁定按钮、行动牌、日志条、进度条、势与状态图标。
- 03_气息系统与六根：气池/气海/锁气/息库/临气区、气骰卡、六根图标、分类牌、骰面、交互标记。
- 04_角色物品地图资料：头像框、角色卡、背包分页、物品图标、场景封面、资料页分页、空状态牌。
- 05_骰子投掷与结果：投掷弹窗、投掷盘、结果牌、按钮、动画关键帧、特效底板。
- 06_清单说明：素材投放清单与风格参考图。

## 使用建议
1. 本包适合做前端接入参考和二次切图，不建议直接把整板整图当 UI 成品贴上去。
2. 优先使用每个分类目录中的拆分素材；具体投放位置参见 `素材投放清单.csv`。
3. 若后续要真正落地到代码层，建议再做一轮：
   - 透明背景化
   - 按钮/徽章/标签单独导出
   - 常用尺寸规范（1x/2x）
   - SVG 或高分 PNG 重绘
